## Projet Symfony 6.4.* de base

Version : **1.0.0**

### Installation

```bash
git clone https://github.com/rcnchris/sf64-base.git my-project-dir
cd my-project-dir
composer app-install
```

### Mise à jour

```bash
composer app-update
```
